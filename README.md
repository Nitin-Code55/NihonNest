# NihonNest 🇯🇵🇮🇳

NihonNest is a dedicated platform designed to simplify the journey of Indian workers transitioning to Japan under the SSW (Specified Skilled Worker) visa program. It provides end-to-end guidance, verified employment placement, and soft-landing solutions to create a transparent and safe community.

## 🚀 Latest Update: The "Cinematic UX" Overhaul (June 2026)

Today's major update brings a complete, top-to-bottom architectural and visual redesign of the NihonNest platform, elevating it to a truly premium, app-like experience. 

### Key Enhancements:
* **Premium Dark Mode Engine**: The entire application has transitioned from a standard light layout to a deep, cinematic dark theme, utilizing a custom Saffron (`#FF9933`) accent palette designed for incredible contrast and readability.
* **Global 3D Parallax Environment**: Implemented a state-of-the-art global background system. As users scroll, the background dynamically pans and zooms (`scale`), creating a deep 3D cinematic effect that reacts to user movement.
* **Glassmorphism UI System**: All primary interface elements—including the sticky navbar, service cards, bundle packages, and contact forms—have been rebuilt using frosted glass logic (`backdrop-filter: blur()`).
* **Dynamic Intersection Observer Animations**: Every section across the site now features scroll-aware animations. Elements fluidly fade and slide into view exactly as they enter the viewport, eliminating sudden visual pops.
* **The "Journey" Timeline Revamp (About Page)**: The static timeline was completely rebuilt into an interactive storytelling experience. An airplane mechanically dives along the central spine based on scroll position, while story cards dynamically slide in from the edges to meet it. 
* **Zero-Lag Physics**: Hover animations on high-transparency glass cards were decoupled from entrance transitions, resulting in an instantly responsive 60fps hover experience devoid of input lag. 

### Core Tech Stack
* **HTML5** (Semantic structuring)
* **Vanilla CSS3** (Custom properties, keyframe animations, glassmorphism filters, cubic-bezier transitions)
* **Vanilla JavaScript** (IntersectionObserver API, RequestAnimationFrame for parallax scrolling)
